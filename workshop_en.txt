[h1] JmcModLib [/h1]
[h2] 1. Overview [/h2]
The main part of this MOD comes from my [url=https://github.com/JMC2002/JmcModLib] prerequisite library of the same name [/url] for Escape from Duckov. It mainly consists of a configuration library, a reflection library, logging library wrappers, and localization library wrappers.

[url=https://github.com/JMC2002/SlayTheSpire2_JmcModLib] GitHub Repository [/url]

[h2] 2. Features [/h2]
[list]
    [*] Provides a settings UI and configuration items that fit the game's own style (including rich-text tooltip boxes), with native controller support
    [img]https://images.steamusercontent.com/ugc/15109940752744817156/70CBEFCB61FFBA0848DCC2DD39EB2DC706FF567A/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [img]https://images.steamusercontent.com/ugc/9668772965325563167/B9E9F375F57885DF3F5AF8625287C06DC7460D54/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] With only a setting_ui.json file, localized configuration items can be scanned and built automatically
    [img]https://images.steamusercontent.com/ugc/9688811441831871705/FAC0BCDE508DA178B3C1BC3B7B8044F03E448845/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] Simply mark the desired control type on a configuration variable to complete all registration. There is no need to pass callbacks, maintain persistent configuration, or maintain localization
    [*] After a key binding is marked as controller-supported, a Steam Input event is registered automatically (a restart is required the first time it is opened). The related localization text follows the same rules as configuration items
    [img]https://images.steamusercontent.com/ugc/12821521369812469438/A13109B265C1CBF7A1BF203BE37388016268136C/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [img]https://images.steamusercontent.com/ugc/16837084861737203949/A2426DF29ECF826F1B40CC28791C923613A44214/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] Provides several game-native prefabs
    [img]https://images.steamusercontent.com/ugc/11103480676072558661/6C3B44A0F2D0176555B87819378A342349AED6E4/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] A reflection library with built-in caching
    [*] Logging library wrappers
    [*] Localization library wrappers
[/list]

[h2] 3. Notes [/h2]
[list]
    [*] Because of the game's mechanics, installing any MOD for the first time can cause save desynchronization. You can use [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747533700] BetterSaveSlots [/url] to solve this
    [*] The localization content of this MOD (including the workshop page and changelog), except Chinese, is generated by ChatGPT. It currently supports all languages officially supported by the game. If you think any wording is inaccurate or would like to contribute your own translation, feel free to open an issue / PR on GitHub or contact me directly
[/list]

[h2] 4. Compatibility and Dependencies [/h2]
[list]
    [*] This MOD depends on Newtonsoft.Json 13.0.4 and Harmony. The former has been published in the release menu
    [*] Since the game is still in EA, this MOD may stop working after game updates
[/list]
